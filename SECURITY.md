# Security policy

## Supported versions

Security fixes are applied to the **current default branch** (`main`). Use the latest commit or released image/tag you trust.

## Reporting a vulnerability

Please report security issues **privately**:

- Use **[GitHub Security Advisories](https://github.com/alsaheem/blogging-mcp/security/advisories/new)** for this repository (preferred), **or**
- Contact the maintainers through a private channel they have published on the repository or org profile.

**Do not** open a public issue, discussion, or pull request that describes an exploitable vulnerability before it is fixed and disclosed.

## What to include

- A short description of the issue and its impact.
- Steps to reproduce or proof of concept, if safe to share.
- Affected versions or commit hashes, if known.

We aim to acknowledge valid reports within a few business days. Thank you for helping keep users safe.
